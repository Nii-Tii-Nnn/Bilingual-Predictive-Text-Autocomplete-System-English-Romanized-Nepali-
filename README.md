
# Bilingual Predictive Text & Autocomplete System

Semester-level Python Computer Engineering project.

Features:
- Trie based autocomplete
- Frequency learning engine
- Heap based Top-K ranking
- Dataset management
- Adaptive learning
- Tkinter GUI
- Analytics dashboard
- Levenshtein fuzzy matching
- N-gram next word prediction

Run:
python main.py

Requirements:
Python 3.10+
No external packages required.
